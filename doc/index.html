Actions

Converter

The "top level" class to perform the conversion. The "real" processing is controlled by the convert_to_markdown method; all other (private) methods perform all kinds of processing on the input text.
c
GitHub

Wrapper around a the Github API using the generic octokit library. It only includes shortcut methods for what is used in the package.
c
GitHubCache

There may be references to different repositories within one IRC log; to avoid re-generating a new GitHub, this wrapper class caches the various instances.

Get the IRC log. The input reference provided in the configuration (i.e., conf.input) is examined whether it is a URL (in which case this is retrieved via HTTP) or not (in which case it is considered to be a local file). Returns a Promise with the content of the input as single string.

get_label

f
get_name_list

Extract a list of nick names (used for present, regrets, and guests, etc) All of these have a common structure: 'XXX+' means add nicknames, 'XXX-' means remove nicknames, 'XXX:' means set them.
f
get_nick_mapping

Get the nickname mapping file (if any). The input provided in the configuration is examined whether it is a URL (in which case this is retrieved via HTTP) or not (in which case it is considered to be a local file). Returns a Promise with the content of the input as an object.
f
handle_scribejs

Handle the 'scribejs' directives. The directives are of the form "scribejs, COMMAND ARGS" or, equivalently, "sjs, COMMAND ARGS".
